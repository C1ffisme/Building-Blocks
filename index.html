Building-blocks

This is a game I made in python. It's not quite completed, but fun at the moment.

View on GitHub Download .zip Download .tar.gz

Building Blocks is a small, two dimensional environment that mimics Terraria. While it is empty, and in need of features, we currently have:

  • Map Generation (With hills, biomes and trees)
  • A 'Survival Mode'
  • A few different kinds of blocks
  • A crafting menu
  • And much more!

Building Blocks is coded in python and pygame, which means even an amateur programmer or bored teenager can easily fiddle around with the code.

As a game, Building Blocks is not complete, however, our Github Page is open for people to submit pull requests and issues. Please help us there!
